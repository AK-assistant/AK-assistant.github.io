# AK Assistant 官网源码

这是 AK Assistant 官方网站的源码仓库，基于 Docusaurus 构建。

本仓库仅用于维护官网相关内容：

- 产品介绍页
- 使用文档
- 更新日志
- GitHub Releases 下载入口

本仓库不是 AK Assistant 桌面软件的完整业务源码仓库。这里发布的是官网与文档内容，不包含桌面应用、服务端或完整业务实现。

## AK Assistant 特性

- 轻快流畅，适合长时间串口调试
- 支持 Windows、macOS 与 Linux
- 统一提供官网、文档与下载入口
- 通过 GitHub Releases 分发安装包

## 本地开发

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
npm run serve
```
